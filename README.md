## Описание

**Анализатор страниц** — веб-сервис для мониторинга сайтов. Позволяет добавлять URL-адреса, проверять их доступность и анализировать SEO-метаданные (заголовки, описания, структуру). Все результаты проверок сохраняются и доступны в истории.

### Возможности

- **Добавление URL-адресов** — добавление сайтов для мониторинга с автоматической нормализацией и проверкой на дубликаты
- **Проверка доступности** — HTTP-запросы к сайтам с получением статус-кодов
- **SEO-анализ** — извлечение метаданных: заголовок (title), первый заголовок (h1), мета-описание (description)
- **История проверок** — хранение всех выполненных проверок для каждого URL с временными метками

## База данных

### Схема

```sql
CREATE TABLE urls (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    created_at TIMESTAMP NOT NULL
);

CREATE TABLE url_checks (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    url_id BIGINT NOT NULL REFERENCES urls(id) ON DELETE CASCADE,
    status_code INT,
    h1 VARCHAR(255),
    title VARCHAR(255),
    description TEXT,
    created_at TIMESTAMP NOT NULL
);
```

## Функциональность

### Основные возможности

- **Добавление URL**
  - Валидация и нормализация URL
  - Проверка на дубликаты
  - Сохранение в базу данных

- **Просмотр списка URL**
  - Отображение всех добавленных URL
  - Показ последней проверки для каждого URL

- **Детальная информация об URL**
  - История всех проверок
  - SEO-метрики

- **Проверка URL**
  - HTTP-запрос к сайту
  - Извлечение title, h1, description
  - Сохранение результатов проверки
